Три недели назад у меня не было ни одного продукта в продакшене. Сейчас их девять. Без команды, без инвестиций, без офиса. Вот каждый из них.

## 1. CoffeeBot — AI-квиз по подбору кофе

**Стек:** Astro, Tailwind, Groq API, Cloudflare Pages

Пользователь отвечает на 8 вопросов о предпочтениях — крепость, кислотность, способ приготовления. AI анализирует ответы и подбирает кофе из базы в 100+ сортов. Нет никакой магии — просто хорошо структурированный промпт и быстрый инференс через Groq.

**Цифры:** 82 уникальных профиля сгенерировано за первую неделю. Время генерации рекомендации — менее 2 секунд.

**Стоимость:** $0/мес. Groq бесплатен, Cloudflare Pages бесплатен.

## 2. InkCloak — детектор и гуманизатор AI-текста

**Стек:** Next.js, DeBERTa LoRA fine-tune, RunPod Serverless, Python, CUDA

Это самый технически сложный продукт. Два компонента: детектор определяет, написан ли текст нейросетью (AUROC 0.9948 на тестовом наборе), а гуманизатор переписывает текст так, чтобы детекторы его не ловили.

Детектор — fine-tuned DeBERTa v3 large. Тренировал на RunPod на A100 (стоило $0.49 за весь процесс). Гуманизатор использует adversarial paraphrasing — генерирует варианты и выбирает тот, который проходит детектор.

**Цифры:** AUROC 0.9948 на 10K тестовых текстов. Inference на P100 — 1.2 секунды на текст до 500 слов.

**Стоимость:** $0/мес в режиме ожидания, ~$0.002 за запрос при нагрузке.

## 3. AICPO — AI-платформа для продуктового исследования

**Стек:** Rails 8, SQLite, Groq, ActionCable, OpenRouter

Это мой основной продукт. Чат с AI-ассистентом, который помогает исследовать рынок: задаёт правильные вопросы, извлекает факты из ответов, строит карту знаний и генерирует 43 стратегических артефакта — от JTBD-карт до финансовых моделей.

Под капотом: двухслойная архитектура с Project Brief (сжатое описание проекта для экономии токенов), Intent Gate (классификация намерений), мультимодельный каскад (Groq для скорости, OpenRouter для качества).

**Цифры:** 43 типа артефактов. Среднее время генерации артефакта — 12 секунд.

**Стоимость:** ~$5/мес (домен + сервер Timeweb).

## 4. Scrutly — бесплатный детектор AI-текста

**Стек:** Astro, DeBERTa, Cloudflare Workers

Упрощённая версия InkCloak — только детекция, без гуманизации. Быстро, бесплатно, без регистрации. Целевая аудитория — преподаватели и редакторы.

**Статус:** в планах, модель уже готова.

## 5. Tool Sites — 199 SEO-страниц на 5 доменах

**Стек:** Astro, Tailwind, MDX, Cloudflare Pages

Пять нишевых сайтов с калькуляторами и инструментами. Контент сгенерирован AI, но каждая страница вычитана и отредактирована. Программатический SEO: шаблон + данные = страница.

**Цифры:** 199 страниц. 5 доменов. Всё на бесплатном тарифе Cloudflare Pages.

## 6. OpenClaw Finance — AI-фабрика финансовых агентов

**Стек:** Python, DeepSeek R1-0528, Docker, SQLite

Пять специализированных агентов: анализ финансовой отчётности, прогнозирование, формирование портфеля, риск-менеджмент, генерация отчётов. Каждый агент имеет свой системный промпт и инструментарий.

**Цифры:** 5 агентов, задеплоено на 2 машинах. DeepSeek R1 обеспечивает качество reasoning на уровне GPT-4 при нулевой стоимости API.

## 7. RackCloud — своя облачная платформа

**Стек:** Claude Opus (проектирование), Proxmox, OpenStack, Python

Архитектура IaaS-провайдера с нуля: 400+ bare-metal серверов, 17 сервисов (compute, storage, networking, billing, dashboard), ценообразование, роадмап на 12 месяцев. Проектировал совместно с Claude Opus.

**Цифры:** 17 IaaS-сервисов, 400+ серверов, 10 стоек. Полная документация за 4 дня.

## 8. DIYCalc — калькуляторы стройматериалов

**Стек:** Astro, Tailwind, MDX, Cloudflare Pages

30+ бесплатных калькуляторов: бетон, плитка, краска, гипсокартон. Каждый считает количество и стоимость материалов. SEO-оптимизация под запросы типа "сколько бетона нужно на фундамент".

## 9. FitCalc — фитнес-калькуляторы

**Стек:** Astro, Tailwind, MDX, Cloudflare Pages

40+ калькуляторов здоровья и фитнеса: TDEE, ИМТ, процент жира, макросы, калории. Научные формулы (Mifflin-St Jeor, Harris-Benedict) с персональными рекомендациями.

## Итог

Девять продуктов. Общая стоимость инфраструктуры: $15/мес (только домены). Всё остальное — бесплатные тарифы Cloudflare, Groq, PostHog, Supabase.

Ключевое: я не писал большую часть кода руками. Factory OS — моя система AI-агентов поверх Claude Code — генерирует, тестирует и деплоит код. Я принимаю продуктовые решения и ревьюю коммиты.

Это не будущее. Это уже настоящее.
